Question: How does configuring the scanner button from the Properties dialog  
box differ from configuring the button from the system tray?  
Answer: Each serves a different purpose. When using the scanners  
Properties dialog box you are instructing the PC which application to activate  
when the scanners button is pressed. The application activated can depend  
on the scanner button number displayed on the scanner.  
When using the system tray (by right-clicking on the application icon displayed  
in the system tray) to configure the button setup for the application, you are  
instructing the application what operation to perform when the button is  
pressed (e.g., Scan to PDF, Scan to email, etc.). The operation performed  
depends on the button number displayed on the scanner.
