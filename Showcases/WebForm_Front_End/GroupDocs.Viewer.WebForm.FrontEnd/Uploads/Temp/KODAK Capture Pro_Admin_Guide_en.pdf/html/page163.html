Page Setup dialog  
box  
Use the Page Setup dialog box to define image scanning parameters,  
automatically delete images, split images or merge images.  
To access Page Setup click File>Page Setup.  
Page setup name  select an existing Page Setup from the drop-down list.  
Icons  
Add: displays the Save As dialog box, which allows you to add a  
new page setup.  
Rename: displays the Rename dialog box which allows you to  
rename the currently selected page setup.  
Delete: displays a confirmation box which allows you to delete  
the selected page setup.
