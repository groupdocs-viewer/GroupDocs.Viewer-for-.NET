If your workstation has internet access:

2

If your workstation does not have internet access:  
1
. From the Capture Pro installation folder (typically Program  
Files>Kodak>Capture Pro), run the License Manager, KCSPLM.exe.  
2
. From the Replace License tab, select Request Replacement. Enter the  
path to where you want the replacement request file to be saved. Select  
Request. A license replacement request file will be created.  
3
. Exit Capture Pro Software.  
4
. Move the replacement request file to a computer with internet access.  
5
. On the computer with internet access, download the License Manager  
tool from the Capture Pro Software download site.
