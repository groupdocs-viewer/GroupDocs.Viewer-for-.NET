N<[2, ^], 20, 3>  
Description  a signed number with a total of 20 characters and 3 digits after  
the decimal point. The number will be extracted from the 2 elements in the  
string, where the elements are separated by the ^ character.  
Example

#
<textformats, width, fracwidth, padding> used to format a number that  
is extracted from a string using textformats and padded with the specified  
character.  
This format is an extension of the format #<textformats, width, fracwidth>  
where the output string will be padded with the specified character.
